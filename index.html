Acerca de Lion Token

Es un token alojado en la red de Binance Smart Chain (BSC) que tiene como objetivo llegar a crear una red de criptomonedas capaces de ser minadas bajo el consenso POW. Este token se encuentra disponible en cualquier wallet descentralizada.

Quema de Tokens

El token se quemará con la quema programada, lo que significa que el valor real se intercambia por los tokens que se envían a la billetera quemada.

Tokenomics

El token se encontrará distribuido en

  • 1% Quema
  • 1% Holders
  • 1% Marketing
